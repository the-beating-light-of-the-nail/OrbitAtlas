# OrbitAtlas

> Space Launch & Rocket Database — tracking rockets, launches, missions, companies, and space programs around the world.

## What is OrbitAtlas?

OrbitAtlas is an English-first, entity-first, SEO-driven space launch database website. It uses structured data (not blog posts) to create long-lasting, high-value pages about:

- 🚀 Rockets
- 🛰️ Launches
- 🏢 Companies
- 🔧 Engines
- 📍 Launch Sites
- 🎯 Missions
- 🇨🇳 China Space Program
- ⚖️ Compare pages

## Tech Stack

- **Nuxt 3** — Vue 3 meta-framework
- **TypeScript** — strict mode
- **JSON** — static data files (no database in MVP)
- **Vercel** — deployment
- **Cloudflare** — DNS/CDN
